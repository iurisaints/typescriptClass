Atividades feitas nesta pasta:

1) Crie uma classe "Carro" com propriedades como marca, modelo, ano e métodos para ligar, desligar e acelerar o carro.

2) Crie uma classe "ContaBancaria" com propriedades como número da conta, saldo e métodos para depositar, sacar e verificar o saldo.

3) Crie uma classe "Pessoa" com propriedades como nome, idade e método para cumprimentar a pessoa.

4) Crie uma hierarquia de classes para modelar animais, com uma classe base "Animal" e classes derivadas como "Cachorro", "Gato" e "Pássaro" com métodos e propriedades específicas para cada tipo de animal.

5) Crie uma classe "Loja" com propriedades como nome, endereço e uma lista de produtos. Adicione métodos para adicionar e remover produtos da lista e para exibir os produtos disponíveis na loja.
